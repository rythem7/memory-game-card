# Memory Card Game

A memory card game built with React. 

## Features
- Randomized card shuffling
- Smooth Animations
- Responsive design

## Technologies Used
- React
- Vite
- JavaScript
- Tailwind CSS
- DaisyUI

## How to Run
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the project: `npm run dev`
